# kartaca-urun-servis

### How To Run It

**Kartaca Ürün ve Servis Yönetimi requires [Docker](https://www.docker.com/) to run.**

```
$ git clone https://github.com/edacicekli/kartaca-urun-servis
$ cd kartaca-urun-servis
$ docker-compose up
```


## Routes
- http://localhost:9000
- http://localhost:9000/ui
